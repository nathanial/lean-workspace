# Display Selected Color Value

**Priority:** High
**Section:** Feature Proposals
**Estimated Effort:** Small
**Dependencies:** None (Tincture.Format already available)

## Description
Show the currently selected color as hex, RGB, and HSL text below the picker.

## Rationale
Users need to see and copy the color value they've selected. Tincture already provides `toHex`, `toRgbString`, `toHslString`.
